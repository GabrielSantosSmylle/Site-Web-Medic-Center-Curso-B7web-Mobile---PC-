------------ Medic Center - Website Responsivo ------------

Projeto desenvolvido durante o curso da B7Web, com foco na criação de um site institucional responsivo para clínica médica fictícia.

O projeto foi desenvolvido para funcionar tanto em Desktop (PC) quanto em Dispositivos Mobile, aplicando conceitos de HTML semântico, CSS moderno e organização de arquivos.

------------ Sobre o Projeto ------------

O Medic Center é um site institucional que apresenta:

Menu de navegação

Banner principal

Seções informativas

Área de notícias

Layout organizado e responsivo

Estrutura separada de CSS e JavaScript

Organização de imagens em pasta própria

------------Tecnologias Utilizadas ------------

HTML5

CSS3

Flexbox

Media Queries

JavaScript

Estrutura de pastas organizada (assets)
